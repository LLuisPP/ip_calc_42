<h2>‚ñ∂Ô∏è Execution</h2>

Use correct syntax, for example:
Single binary octet:
`````
./ipcalc -b 11100000
`````
IP binary octet:
`````
./ipcalc -b 11100000.00000000.00000000.0000000
`````
Single decimal number (0-256):
`````
./ipcalc -d 116
`````
Decimal IP:
`````
./ipcalc -b 192.168.94.14
`````
